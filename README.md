Download the latest version from our [Releases](https://github.com/joseook/leetcode-ghost-joseok/releases) page:

## Usage

1. **Launch the application**: It will start in Ghost Mode by default
2. **Enter your OpenAI API Key**: Required for AI processing of screenshots
3. **Navigate to a LeetCode problem**
4. **Capture screenshots**: Use keyboard shortcuts to capture the problem
   - `Ctrl+Shift+1` (Windows/Linux) or `‚åò+‚áß+1` (macOS): Capture first screenshot
   - `Ctrl+Shift+2` or `‚åò+‚áß+2`: Capture second screenshot (if needed)
   - `Ctrl+Shift+3` or `‚åò+‚áß+3`: Capture third screenshot (if needed)
5. **Process screenshots**: Get AI-powered solution suggestions
   - `Ctrl+Shift+A` or `‚åò+‚áß+A`: Process with 1 image
   - `Ctrl+Shift+B` or `‚åò+‚áß+B`: Process with 2 images
   - `Ctrl+Shift+C` or `‚åò+‚áß+C`: Process with 3 images
6. **Window Management**:
   - `Ctrl+Shift+H` or `‚åò+‚áß+H`: Hide window
   - `Ctrl+Shift+S` or `‚åò+‚áß+S`: Show window
   - `Ctrl+Shift+Q` or `‚åò+‚áß+Q`: Quit application
   - `Ctrl+Shift+M` or `‚åò+‚áß+M`: Toggle window drag mode

You can also access all functions through the system tray icon.
